Funko Potter

As part of my Computer Graphics exam for the Master's in Computer Engineering at PoliTo, I completed a project that consisted of two parts: the first part involved using Blender to create a 3D render using various modeling techniques. In the second part, I designed an interactive game using HTML and X3DOM graphics library. I was given the task to design the game experience, coming up with a "dodge/attack" with single or multiplayer modes. I also worked on the characters implementation and their skills, with a focus on the balance of the game. Moreover, I had the task to implement the gameplay, making sure it was fully working in a satisfying manner.
